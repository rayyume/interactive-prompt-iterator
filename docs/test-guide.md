# 测试指南

## 快速测试步骤

### 1. 启动应用
```bash
npm run dev
```

### 2. 测试新对话功能
- 发送一条消息创建对话
- 点击"新对话"按钮
- ✅ 检查: 侧边栏历史记录应该保留

### 3. 测试收藏页新对话
- 切换到"收藏"标签页
- 点击"新对话"按钮
- ✅ 检查: 应自动切换到"对话"标签页

### 4. 测试文件上传布局
- 上传2-3个文件
- ✅ 检查: 文件卡片显示在输入框上方
- ✅ 检查: 输入框内只有上传按钮,不被挤占

### 5. 测试收藏按钮位置
- 生成一个提示词
- ✅ 检查: 收藏按钮在卡片标题右侧
- 切换"预览视图"和"结构详情"
- ✅ 检查: 收藏按钮位置保持一致

### 6. 测试全屏预览
- 点击放大按钮
- ✅ 检查: 预览窗口占据90%屏幕宽度

### 7. 测试表单宽度
- 查看优化方向建议表单
- ✅ 检查: 表单宽度固定,不会太窄

## 自动化测试
```bash
node scripts/tests/test-7-fixes.js
```
